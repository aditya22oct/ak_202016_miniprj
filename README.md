# ak_202016_miniprj
AAMR Course Mini Project Task

Clone the repository in the your catin_ws/src folder. 

After that do 

```
roscd 
cd ..
catkin_make
source ~/<catkin workspace name>/devel/setup.bash
```
and then to run the package using command.

`roslaunch miniprj start.launch`
